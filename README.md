## Прогноз эффективности химических соединений против вируса гриппа

В рамках проекта была решена задача прогнозирования эффективности химических соединений против вируса гриппа на основе набора молекулярных дескрипторов. Данные содержали информацию о 1000 соединений и включали 210 числовых признаков, характеризующих структуру молекул. Параметры, характеризующие эффективность, обозначаются как IC50, CC50 и SI.


В ходе работы был выполнен разведочный анализ данных, заполнены пропуски медианными значениями, удалены высококоррелированные признаки и выполнено логарифмирование целевых переменных.

Построены модели машинного обучения для решения задач:


регрессии IC50, CC50, SI;

классификации превышения медианных значений IC50, CC50 и SI;

классификации превышения порога SI > 8;

Репозиторий содержит ноутбуки ipynb для анализа данных и решения каждой задачи.


Полученные модели могут использоваться для предварительной оценки эффективности химических соединений.
